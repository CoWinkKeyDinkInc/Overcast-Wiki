Commands

All the commands and how to use them

  • /Join
  • /Toggle
/Join Usage:

/join [team]

About:

This is how you join the game

Other info:

You can only specify a team if you have donated or are the maker of the map currently playing.

/Toggle Usage:

/toggle [setting]

About:

This changes settings specific to you.

Blood:

See blood when a player gets hurt

Death Messages:

Death messages displayed to you

Join Messages:

Join messages displayed to you

Observers:

See other observers while spectating

Picker:

Open a helpful GUI for picking classes and teams

Punishment Messages:

Punishment messsages displayed to you

Scoreboard:

See the scoreboard with game info

Timer:

Show remaining time at the top of the screen
